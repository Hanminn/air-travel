# Air Travel

# Overview
여행지에는 관광지가 많아 여행 계획이 복잡하고, 기존 플랫폼은 사용자의 취향을 충분히 반영하지 못합니다. Air Travel을 활용하면 맞춤형 여행 일정 추천과 관광지 설명을 제공하여, 효율적이고 개인화된 여행 경험을 제공할 수 있습니다.

# 주요기능
## 서비스 소개
- 자유 여행을 떠날 때 여행 계획을 짜기 힘들어 하는 사람들을 위한 자유여행을 패키지 여행스럽게 일정을 짜주는 AI 여행 스케줄 서비스

## 프로젝트 기능
- 함께 여행하는 친구들과 채팅
  - 생성된 이미지를 친구들과 채팅방에서 공유하며 같이 수정할 수 있다.
- AI를 활용한 여행일정 생성
  - 여행 생성 페이지를 통해 사용자에 맞는 여행 일정을 생성해 준다.
- 실시간 번역과 통역
  - 여행 중 언어의 장벽을 낮추기위해 번역페이지에서 실시간 통역과 번역이 가능하다.
- 가이드를 대신하는 AI카메라
  - 여행 중 관광명소, 랜드마크에 대해 궁금한 것이 있으면 AI카메라 페이지에서 촬영해 랜드마크에 대한 정보를 받을 수 있다.
- AI가 완성한 그림일기
  - 여행을 마치고 일기를 작성하면 그 일기 내용에 맞게 AI가 이미지를 만들어준다.
