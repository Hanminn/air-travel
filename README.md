# Air Travel

# Overview
여행지에는 관광지가 많아 여행 계획이 복잡하고, 기존 플랫폼은 사용자의 취향을 충분히 반영하지 못합니다. Air Travel을 활용하면 맞춤형 여행 일정 추천과 관광지 설명을 제공하여, 효율적이고 개인화된 여행 경험을 제공할 수 있습니다.
